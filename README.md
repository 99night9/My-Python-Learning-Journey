My-Python-Learning-Journey
这是一个记录我 Python 学习过程的项目仓库。目前包含两个我本学期完成的 Python 程序，分别用于心理压力测评数据分析和学习状态评估。

项目简介
设计题目一：心理压力测评数据分析
该程序用于收集 8 名被试的压力得分，进行统计分析，输出平均分、压力程度分布、最高/最低分，并给出整体建议。

设计题目二：学习状态评估系统
该程序根据用户输入的学习时间、作业完成率和课堂表现，评估学习状态，并给出相应的改进建议和鼓励语句。

如何运行
环境要求
Python 3.2 或以上版本

运行步骤
1.克隆本仓库到本地：

git clone https://github.com/你的用户名/My-Python-Learning-Journey.git
进入项目目录，运行任一程序：

python 设计题目一.py
或

python 设计题目二.py
按照提示输入相应数据即可查看分析结果。

使用列表存储压力得分

使用循环和条件判断进行数据验证和分类统计

输出包括平均分、分布情况、极值和建议

学习心得与规划
在浏览 GitHub 上其他 Python 项目时，我深刻感受到规范的项目结构、清晰的文档和持续的代码维护对于展示个人技能的重要性。通过本次任务，我学会了如何使用 Git 管理代码版本、如何撰写规范的 README 文件，以及如何将本地项目与远程仓库关联。

这些技能不仅帮助我更好地组织个人项目，也让我意识到持续记录和展示学习过程的价值。未来，我计划利用这个仓库：
